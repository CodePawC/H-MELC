## 主干：
```

```

---

针对：

  * 辐射
  * 特种设备
  * 压力容器
  * 强检设备


---
## 示例：
### 放射设备：
```

```

---
### 压力容器：
```
建档→ 注册→ 定检→ 安全阀校验→ 压力测试→ 整改
```

---
# 第三层：智能分析流程
AI主动触发。
---
## 示例：
```
高故障率→ AI分析→ 风险预警→ 建议停用→ 生成整改建议→ 提交设备科
```

# 三、为什么必须“流程引擎化”
因为医院流程：
# 一定会变化。
例如：
某院：
  * 维修金额 > 5000
    * 需要财务审核


另一院：
  * > 10000


如果写死：
后期一定灾难。
