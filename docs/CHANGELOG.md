# Changelog

## v0.1.0 · 20260527.017 - H-UMDG 组织与人员主数据

- 组织与人员主数据接入 H-UMDG：新增院区、完善科室、预留人员与学科主数据，OS 设备档案与相关业务字段统一调用 H-UMDG 科室/人员/学科选择器。

## v0.1.0 · 20260527.016 - H-UMDG 分类目录选择器

- 设备档案新增/编辑接入 H-UMDG 医疗器械分类目录主数据选择器，支持从 H-UMDG 实时搜索、选择并绑定分类目录。
- 新增设备升级为智能建档：支持铭牌/附件资料上传、AI/OCR 信息抽取、H-UMDG 医疗器械分类目录自动匹配、人工审核确认和正式建档流程。

## v1.1.0 - 一期闭环收敛与 H-UMDG 外部主数据接入

- 明确 H-UMDG 是独立的医院统一主数据治理平台。
- 明确 H-MELC 不维护设备分类、设备标准名称、医疗器械分类目录和厂商机构权威库。
- 新增 H-UMDG 外部 API 代理接口。
- 新增 H-UMDG 只读字典缓存。
- 新增设备标准名称候选申请。
- 新增厂商机构候选申请。
- 设备台账新增 H-UMDG 装备字典和厂商机构引用字段。
- 保留已验证的设备档案、抽屉和标签打印能力。
